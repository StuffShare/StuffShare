How to use StuffShare


1) Registration / Logging In

  • To access StuffShare's features, you must first login. If you do not have an account, click on "Sign Up" located in the top right.

    • 2) Adding Items

  • To add an item for sharing, click on "Add Item in the right menu bar
  • Fill out all the form fields:
    • Item Name - Name of the item
    • Notes - Addition information on the item
    • Quality - The item's condition
    • Location - Enter the item's location
    • Visibility - Choose between public or private
    • Return Date - Select a date of which you want this to be returned by
    • Picture - Upload a picture of the item

    3) Editing Items

  • Click on "Edit/Delete Item" to view the list of items you've shared.
  • You can edit or delete the item by clicking the "Edit" or "Delete" button to the right of the table.

    • 4) Private vs Public lists

  • Public items can be viewed by all users on the site.
  • Private items can only be viewed by friends.

    • 5) Adding Friends

  • To add friends, click on "Add friend" in the right menu.
  • Select the Friend ID of the user you wish to add and click submit.
  • Friend ID's can be seen through clicking the "Get Friend Code" link.

    • 6) Accepting Friend Requests

  • Click on "Friend Requests" to view all pending friend requests.
  • To accept a Friend Request, click "Accept".
  • Similarly, you can decline a pending friend request by clicking "Decline".
